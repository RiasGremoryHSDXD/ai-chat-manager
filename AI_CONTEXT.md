# AI Assistant Project Context

## Project Structure & Deployment
- **Type**: Chrome Extension (React + Vite).
- **Build Output**: `dist/` folder.
- **IMPORTANT**: The chrome extension loads code from the `dist/` folder.
- **Rule**: Anytime you modify code in `src/`, you **MUST** run `npm run build` to compile the changes into `dist/`. Without this, changes will not be visible in the browser.

## Commands
- **Build**: `npm run build`
